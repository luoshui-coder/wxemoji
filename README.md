# 表情包工坊 Meme Studio

一个轻量的表情包生成 & 处理工具，支持两种模式：**AI 一键生成** 和 **上传素材手动裁切**。基于 Flask，单文件部署，开箱即用。

---

## 两种工作模式

### 模式一：AI 生成（需要 API）

输入角色描述（可选上传参考图），AI 自动生成 24 格表情包网格、横幅图、App 图标及中文文案，生成后可在裁切工坊微调分割线。

**适合场景**：从零创作一套全新表情包

```
输入描述 → AI 生成 4×6 网格 → 自动切片 → 裁切微调 → 打包下载
```

### 模式二：上传素材手动处理（无需 API）

已有表情包网格图？直接上传，自动识别分割线并切片，支持 4×6（24 格）和 4×4（16 格）两种布局。也可手动上传横幅图、图标等素材，统一打包下载。

**适合场景**：处理已有的表情包素材、调整裁切位置、打包资源

```
上传网格图 → 选择布局（4×6 / 4×4）→ 自动切片 → 裁切微调 → 打包下载
```

> 模式二不需要配置 API Key，安装依赖后直接启动即可使用。

## 快速开始

### 1. 克隆项目

```bash
git clone <repo-url>
cd wxemoji
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 启动

```bash
python app.py
```

浏览器打开 http://localhost:8080 即可使用。

如果只用「上传素材」模式，到这一步就结束了。

---

## 配置 API（仅 AI 生成模式需要）

```bash
cp config.yaml.example config.yaml
```

编辑 `config.yaml`，填入你的 API Key 和接口地址：

```yaml
openai_api_key: "sk-xxxx"
openai_base_url: "https://your-api-endpoint/v1"
```

也支持 `.env` 文件或环境变量，优先级：`config.yaml` > `.env` > 环境变量 > 默认值。

> 不配置 API 也能正常使用「上传素材」模式，只是无法使用 AI 生成功能。

### 浏览器端覆盖

点击页面右上角 ⚙ 图标，可在浏览器端单独设置 API Key 和 Base URL（保存在 localStorage），优先级高于服务端配置。适合多人共用同一服务、各自使用不同 API Key 的场景。

## 使用说明

### AI 生成

1. 左侧输入角色描述（如：一只慵懒的橘猫，圆眼睛，穿着睡衣）
2. 可选上传参考图，AI 会以此为原型重绘
3. 点击「开始生成」或 `Ctrl+Enter`
4. 等待 2-5 分钟，生成完成后自动展示结果

### 上传素材

1. 切换到「上传素材」标签页
2. 上传网格图时，选择布局：**4×6**（24 格）或 **4×4**（16 格）
3. 点击「自动切片 & 进入裁切工坊」
4. 拖拽红/蓝线微调裁切位置，确认后更新结果

### 裁切工坊

- 红色横线 = 行分割，蓝色竖线 = 列分割
- 拖拽手柄调整位置，松手自动对齐像素边界
- 「重置」恢复均分，「确认裁切」提交更改

### 下载

- 「下载全套资源」→ ZIP 包（stickers/ + banner + logo + info.txt）
- 横幅和图标也可单独保存

## API 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/generate` | POST (SSE) | AI 生成表情包套装，流式返回进度 |
| `/api/slice` | POST | 自动检测网格线并切片 |
| `/api/crop` | POST | 使用精确像素坐标裁切 |
| `/api/download-zip` | POST | 打包资源返回 ZIP |
